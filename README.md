# 知乎
纯HTML+CSS3实现的仿知乎登陆页面<br />
<br />

### 前言
这是2019年9月花了一天时间创作的一个小demo，当时看到知乎的登陆页面后觉得很有设计感，就从官网下载了部分图片素材，自己尝试做了一个还原度比较高的页面。（附：固定背景是用```background-attachment: fixed```实现的。
<br />）
